# Народный Фронт — Design System

Дизайн-система для **Общероссийского народного фронта (ОНФ) / Народного фронта «За Россию»** — всероссийского общественного движения, публичная работа которого сосредоточена на контроле за исполнением поручений президента, обращениях граждан (Прямая линия), общественном надзоре за госзакупками и волонтёрской мобилизации.

Тон системы — церемониальный, бюрократически-официальный и подчёркнуто числовой: почти каждое утверждение опирается на конкретный показатель, а число становится главным героем макета.

---

## Визуальная тема

Система угловатая, плотная и декларативная. Гигантская конденсированная типографика в ВЕРХНЕМ РЕГИСТРЕ, единственный красный акцент, прямые углы и агрессивное использование пустого пространства. Никаких градиентов, размытий, скруглённых карточек или декоративных иконок — бренд опаловый и заявительный.

Ключевые принципы:

- **Число — это герой.** Контраст между одним огромным числом и мелкой подписью к нему. Стат-слайд на 80% пуст.
- **Угловатость.** Прямые углы (0–2px). Скругления (999px) только на мелких мета-чипах. Карточки 12–24px радиуса читаются вне бренда.
- **Конденсированный верхний регистр.** Длинные русские заголовки умещаются в одну строку благодаря узким буквам.
- **Один акцент.** Глубокий красный `#C00000` — единственный акцент в современной работе. Триколор 2011 года используется только в логотипе.
- **Кавычки-ёлочки.** Всегда «…», никогда "…" или „…". Крупные декоративные « » — фирменный мотив вокруг цитат.
- **Без эмодзи.** Никогда не уместны.

| `--color-blue-light` | `#00A4D8` | Светло-синий |

### Нейтральные

`--color-ink #3B3838` · `--color-gray-90 #1F1F1F` · `--color-gray-70 #4A4A4A` · `--color-gray-50 #7A7A7A` · `--color-gray-30 #BFBFBF` · `--color-gray-15 #E0E0E0` · `--color-gray-8 #F0F0F0` · `--color-paper #F7F6F4` (тёплый off-white).

**Рекомендация:** используйте палитру 1 для продуктовой/деко/UI-работы; палитру 2 — только для логотипа и легаси-контекстов. Они сочетаются чисто, потому что красные близки.

### Фоны

- **Красный геометрический фон** — фирменный. Перекрывающиеся плоские красные треугольники в трёх тонах, эффект «флага в движении». Для титульных слайдов и разделителей. Воспроизводится процедурно классом `.nf-bg-flag`; растровая версия — `assets/bg_red_geometric.jpg`.
- **Бумага (`#F7F6F4`)** или чистый белый — для контентных слайдов.
- **Навигационный синий (`#004B92`)** — изредка для цитат / портретных слайдов.

## Типографика

- **Дисплейный / заголовочный шрифт:** **DIN Pro Condensed** — Black для крупного, Bold для среднего, Medium для меток, Light для дисплейного текста. Узкие буквы вмещают длинные русские заголовки в одну строку.
- **Основной текст:** **DIN Pro** (прямое начертание) — гуманистический контрапункт к конденсированному дисплейному шрифту.
- **Моно:** системный стек (`ui-monospace, SF Mono, Menlo, Consolas`).
- **Масштаб:** сильно смещён в сторону дисплея. Геройские числа достигают 148px. Основной текст — скромные 16px. Контраст — это и есть смысл.

| Токен | Размер | Применение |
|---|---|---|
| `--fs-giant` | clamp(64–148px) | Геройское число |
| `--fs-h1` | clamp(44–96px) | Заголовок раздела |
| `--fs-h2` | clamp(32–60px) | Подзаголовок |
| `--fs-h3` | clamp(24–40px) | — |
| `--fs-lead` | 22px | Лид-абзац |
| `--fs-body` | 16px | Основной текст |
| `--fs-small` | 14px | Подписи |

Веса: Light 300 · Regular 400 · Medium 500 · Bold 700 · Black 900.

### Правила набора

- **Регистр.** Заголовки / статистика / разделители: ВЕРХНИЙ РЕГИСТР. Даты, площадки, подписи: обычный регистр.
- **Числа.** Тонкий пробел как разделитель тысяч (`881 438`, `5 382`). Единицы измерения — отдельной строкой под числом.
- **Акцент.** Жирный (DIN Black/Bold) несёт акцент; курсив не используется. Красная подсветка — на единичном важном слове или числе внутри заголовка.

## Принципы

### Сетка и раскладка

- 12-колоночная сетка, отступы 80px на слайдах, 24px в UI.
- Щедрый левый отступ для заголовков — они начинаются в ~80px от края, никогда не центрируются.
- Номер слайда — снизу справа в красном чипе; тонкая красная линейка под заголовком; логотип снизу слева только на обложках.

### Движение

- Деки статичны. UI-движение минимально и уверенно: `120–200ms` ease-out фейды/сдвиги, без отскока и пружин. Фейд + сдвиг на 4px — дефолтный вход.

### Hover / press / focus

- **Hover:** затемнение фона на ~8% (`#C00000` → `#A60000`); ссылки `#C00000` → `#8B0000`.
- **Press:** трансформ scale 98% + более тёмный фон.
- **Focus:** красное кольцо 2px, отступ 2px.

### Границы и тени

- Преимущественно **1px hairline** `#E5E5E5`, либо 2px `#111` для разделения заголовочных зон.
- Толстые красные полосы (6–12px) — декоративные разделители под заголовками.
- Тени экономны: одна мягкая карточная тень `0 2px 8px rgba(17,17,17,0.10)`.

### Прозрачность / блюр

- Не используются. Бренд опаловый. Не накладывайте размытые стеклянные панели.

## Компоненты

| Компонент | Описание |
|---|---|
| **Кнопки** | Прямоугольные (радиус 0–2px), красная заливка/контур, верхний регистр, конденсированный шрифт. Hover — темнее на 8%. |
| **Карточки** | Белый прямоугольник, радиус 0–2px, граница 1px `#E5E5E5`, малая тень. Тонкая красная полоса сверху или слева. Заголовок внутри — конденсированный верхний регистр. |
| **Стат-карточка** | Гигантское число (`--fs-giant`) + мелкая подпись под ним. На 80% пустая. |
| **Пулл-квоты** | Цитата в «ёлочках», обрамлённая крупными декоративными красными « » (до 120pt). Атрибуция — площадка + дата ниже. |
| **Топик-плитки** | Прямоугольные плитки тем с конденсированными заголовками. |
| **Инпуты** | Прямоугольные, граница 1px, focus — красное кольцо 2px. |
| **Шапка / навигация** | Логотип + конденсированная навигация в верхнем регистре. |
| **Чип номера слайда** | Красный пилл с белым числом, снизу справа. |

### Иконография

Исходный дек **не использует декоративных иконок**. Визуальная коммуникация несётся гигантской типографикой, логотипом (триколорная V), документальной фотографией и крупными «ёлочками».

- **Предпочтительно:** **Tabler Icons** — линейные, штрих 1.5px, квадратные окончания. ⚠ Флаг-замена — первичного набора ОНФ не существует.
- **Вторично:** **Lucide**.
- **Никогда:** эмодзи, 3D-иконки, дуотон, скруглённые окончания, иллюстративная иконография.
- Все иконки в `currentColor`, размер по умолчанию 20px, штрих 1.5px.

### Логотипы

- `assets/logo_onf.svg` — полная эмблема (триколорная V + диагональный леттермарк). На белом/светлом.
- `assets/logo_onf_mono.svg` — монохром, для малых размеров.
- `assets/mark_v.svg` — голая V-марка, где леттермарк нечитаем (< 48px).
- `assets/bg_red_geometric.jpg` — фирменный угловатый красный фон.

## Флаг-замены

- **DIN Pro Condensed** — основной дисплейный шрифт (файлы в `fonts/`, подключены через `@font-face` в `colors_and_type.css`). Если лицензии нет, ближайшая свободная замена — **Barlow Condensed** (Google Fonts).
- **Набор иконок → Tabler Icons.** Первичной библиотеки в исходнике нет. При публикации внутреннего набора — положить SVG в `assets/icons/`.
